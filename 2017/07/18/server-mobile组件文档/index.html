按钮

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
type 按钮类型 string 否 primary,default,disabled primary
name 按钮文字 string 否 提交

输入框

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
value 输入框绑定值(v-model方式绑定) String,Number 否
format 输入框样式 String 否 input,textarea input
rows 输入框显示行数(format="textarea"时生效) Number 否 8
disabled 输入框是否能够输入 String,Boolean 否
placeholder
type text

提示消息

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
type 文字提示类型 String 否 warning,error warning
icon 自定义图标 String 否
text 说明文本，当文本为空时，消息不展示 String 无

弹窗

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
visible 控制弹窗是否展示 Boolean 否 true/false false

图片预览(已全局配好)

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
src 图片地址 String 是
alt 图片替代文本(图片无法显示时展示) String 是
isShow 是否开启预览 Boolean 是 true/false

TabContainer

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
tabs tab名称 Array 是 []

分享给朋友

使用方式:

API

参数名 说明 类型 是否必须 可选值 默认值
visible 是否展示遮罩 Boolean 是 true/false false
title 分享标题 String 与公众号名称相同
cid 课程cid String 是
teacher 教师名 String 是
course 课程名 String 是
link 分享链接 String
imgUrl 分享图标 String
type 分享类型 String music,video,link link
dataUrl 分享数据链接(用于music、video类型) String
success 确认分享回调 Function
cancel 取消分享回调 Function
